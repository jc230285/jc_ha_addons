# Home Assistant Add-on: MariaDB

MariaDB database for Home Assistant with optional Galera clustering support.

![Supports aarch64 Architecture][aarch64-shield] ![Supports amd64 Architecture][amd64-shield] ![Supports armhf Architecture][armhf-shield] ![Supports armv7 Architecture][armv7-shield] ![Supports i386 Architecture][i386-shield]

## About

You can use this add-on to install MariaDB, which is an open-source (GPLv2 licensed) database. MariaDB can be used as the database backend for Home Assistant. This add-on also supports Galera clustering for high availability setups with multiple Home Assistant instances.

## Features

- MariaDB database server
- Optional Galera clustering for high availability
- Automatic database and user management
- Compatible with Home Assistant recorder integration
- Support for phpMyAdmin add-on
